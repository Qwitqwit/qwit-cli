## [Documentation](DOC.md)

## Install the cli

### Cargo

```bash
    cargo install qwit
```

### Code

```bash
    git clone https://github.com/Qwitqwit/qwit-cli
    cd qwit-cli
    cargo build --release
    cargo install --path .
```

## Usage

```bash
    qwit --help
```
